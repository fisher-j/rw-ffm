1  Data description

The following is generated from the field_descriptions table which is included in the database which can be downloaded here.

First, get table as a dataframe.

con <- DBI::dbConnect(RSQLite::SQLite(), "../data/rw.db")
DBI::dbListTables(con)
 [1] "clumpsaplings"      "collectcrew"        "collectdatasheets" 
 [4] "collectdates"       "collections"        "cwd"               
 [7] "datasheets"         "expand_collection"  "field_descriptions"
[10] "fuelmetadata"       "fwd"                "plots"             
[13] "reftrees"           "regencounts"        "regenheights"      
[16] "regenmetadata"      "stages"             "station"           
[19] "transects"          "treedefect"         "treemetadata"      
[22] "trees"             
descriptions <- tbl(con, "field_descriptions") |> collect()

Then print out the table in sections.

print_desctiption <- function(table_name) {
  cat("\n\n## `", table_name, "`\n\n", sep = "")
  filter(descriptions, table == table_name) |>
    select(-table) |>
    knitr::kable() |>
    print()
}
walk(unique(descriptions$table), ~ print_desctiption(.x))

1.19 station

field description
transectid FK Refers to transects.transectid.
metermark Meters from plot center along the transect where the measurement was taken.
depth Combined depth of duff and litter. This was measured to the top of the litter layer and may include 1-hour fuels.
pctlitter Percent of duff and litter depth that is composed of litter. Litter was judged by having a lighter color, and particles that had not yet begun to break down, for instance, redwood sprays with leaves still attached.
fbd Fuel bed depth was measured as an ocular estimate of the average combined depth of fine and coarse woody fuels in a 1 m radius circle, centered at the metermark. This excluded duff.
livewoody Percent cover of live woody material within a 1-meter radius sampling cylinder, 2-meters tall. Alternatively, Percent of ground that would be completely covered if all woody vegetation within the sampling cylinder were pushed to the ground.
deadwoody Same as for live woody, but for dead material attached to live plants.
woodyheight Average maximum height of live or dead woody vegetation within the sampling cylinder. If a sheet were draped over the woody vegetation only, the average height of the sheet.
liveherb Percent cover of live herbaceous vegetation, as for livewoody.
deadherb Percent cover of dead herbaceous vegetation, as livewoody.
herbheight Average maximum height in meters, as for woodyheight.

decayclass Decay class 1-5, 1=Fine branches still present, 2=Some branches and bark missing, 3=most branches and much bark missing potentially minor decay, 4=Significant decay, 5=Almost completely rotten.
